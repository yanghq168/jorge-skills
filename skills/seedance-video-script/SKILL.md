# Seedance 2.0视频剧本工作流

本skill定义了为Seedance 2.0生成5分钟视频剧本的完整多Agent协作流程。

## Seedance 2.0简介

Seedance 2.0是字节跳动推出的AI视频生成工具，支持：
- 文生视频：通过文本描述生成视频
- 图生视频：基于图片生成动态视频
- 视频时长：最长可生成5分钟视频
- 画质：支持1080P高清输出

## 工作流概览

```
需求分析Agent → 剧本结构Agent → 分镜设计Agent → 提示词生成Agent → 审核优化Agent
      ↓              ↓              ↓              ↓              ↓
  理解视频需求   设计5分钟结构   设计每镜画面    生成Seedance   检查完整性
                                              可用提示词
```

## 使用方式

1. 用户提供视频主题、风格、目标受众等信息
2. 指挥官（权权管家）读取本skill中的YAML工作流定义
3. 需求分析Agent深入理解视频需求
4. 剧本结构Agent设计5分钟视频的整体结构
5. 分镜设计Agent细化每个镜头的画面描述
6. 提示词生成Agent将分镜转化为Seedance可用提示词
7. 审核优化Agent检查剧本完整性和可行性
8. 输出可直接用于Seedance 2.0的完整剧本包

## 快速示例

**用户输入**: "帮我生成一个5分钟的产品介绍视频剧本，主题是智能手表，风格科技感，目标受众是年轻职场人"

**指挥官执行流程**:
1. 需求分析Agent：明确视频目标、受众画像、核心信息、风格调性
2. 剧本结构Agent：设计5分钟结构（开头30s-发展3min-高潮1min-结尾30s）
3. 分镜设计Agent：设计20-25个分镜，每个镜头画面、动作、时长
4. 提示词生成Agent：为每个分镜生成Seedance提示词（英文、详细描述）
5. 审核优化Agent：检查剧本完整性、提示词质量、时长准确性
6. 输出完整剧本包（含剧本、分镜表、提示词列表、使用说明）

## 关键原则

- **时长精准**: 严格控制5分钟时长，每个分镜精确到秒
- **画面可生成**: 确保每个分镜的画面描述可被AI理解和生成
- **提示词优化**: 针对Seedance 2.0优化提示词格式和用词
- **叙事完整**: 保证剧本有清晰的开头-发展-高潮-结尾结构
